Ligg.EasyRPADesk

Easy codeless Windows programming frameworklatest version: 3.5.3

Ligg.EasyRPADesk is a . Net/c # based, codeless Windows application programming framework and UI library. By it, never need coding, only through text and data files in an Excel formular & building blocks like manner, can build any complex UI, and real time deploy and execute. Suitable for RPA, host computer, O&M automation, Mes and other systems, as well as for web clients. By this framework,you can do as following you can do as following

To build any complex Windows& Console app UI; to achieve functionality expansion by dynamically loading 'Plug and Play' .Net& COM Core Business Logic processing (CBLP) components or Outer-embeded Standard Servide(OeStdservice) Components ; supports multi-languages, and various executing modes like Sync/ Async/ Async Awaited/ Managed ThreadPool.

By Shell scripts, to implement process control (value assignment, conditional judgment, loop, jump, etc.); to realize basic functions (string/ file handling, logic/ mathematical operation, data input/ output, data conversion, encryption/ decryption, input verification, form field validation, etc.); to execute Windows script and Python script.

With interacting with UI elements, by inserting the outer-embedded scripts(OeShell) at any point, to interpret, deploy and execute in real time, thus realize system expansion and Robotic Process Automation.
