Ligg.EasyWinApp

Easy codeless Windows programming frameworklatest version: 3.5.3

It is a Windows application programming framework and UI library, includding 2 parts of WinformApp and ConsoleApp. By this framework, never need any code, only through data files(including UI config and scripts-so called Ligg Shell-including UI Shell and OeShell, support formats of .xlsx, .csv, .xml and .json), you can do as following.

To build any complex Windows/ Console app UI, in an Excel formular like manner; to achieve functionality expansion by dynamically loading 'Plug and Play' .Net/ COM Core Business Logic processing (CBLP) component or Outer-embeded Standard Servide(OeStdservice) Component ; supports multi-languages, and various executing modes like Sync/ Async/ Async Awaited/ Managed ThreadPool.

By Shell scripts, to implement process control (value assignment, conditional judgment, loop, jump, etc.); to realize basic functionalities (string/ file handling, logic/ mathematical operation, data input/ output, data conversion, encryption/ decryption, input verification, form field validation, etc.); to execute Windows script and Python script.

With interacting with UI elements, by inserting the outer-embedded scripts(OeShell) at any point, to interpret, deploy and execute in real time, thus realize system expansion and RPA(Robotic Process Automation).
